## Installation

1. Clone the repository:

```bash
git clone https://github.com/codexahmar/Islamic-App
```

2. Navigate to the project directory:

```bash
cd Islamic-App
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
